## 為什麼要投票

在現今民主的社會裡，投票表決是一種能夠展現民主的方式。除此之外，還能夠分析各類的意見，做出更準確的抉擇

## 為什麼使用投票指令

- 能夠使用一行指令建立完整的投票
- 能夠自定義投票選項
- 完全匿名的投票結果
- 完整的投票分析數據

## 如何開始使用投票指令

1. 首先先 [邀請我](/blog/how-to-invite-bot) 到你的伺服器
2. 使用 poll 指令

   ```
   &poll 班長投票 小明 小智 曉欣
   ```

   ![使用投票指令輸入範例](@static/blog/new-poll-command/step-usage-1.png)

3. 這樣你就成功建立一個投票了

   ![使用投票指令完成](@static/blog/new-poll-command/step-usage-2.png)

恭喜你就成功邁出民主的第一步了!
每人都可以點擊下方的按鈕投票
如果要結束投票可以點擊 "結束投票" 按鈕
