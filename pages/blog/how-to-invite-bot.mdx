在現今兩大 Discord 音樂機器人 Rythm 及 Groovy 遭到 YouTube 封殺後，大家都在尋找能夠代替的機器人，YEE 式機器龍能免費讓你就能享受到最棒的全中文音樂機器人體驗，這篇文章將會教你它的指令及用法！

## 什麼是 Discord 機器人？

機器人在 Discord，這個讓大家透過文字和語音交流的平台上，扮演著很重要的角色，透過使用簡單的「指令」來跟機器人互動，幫助你自動化的管理 Discord 伺服器，或是完成許多瑣碎的事情。

通常都可以歡迎新成員，封鎖製造問題的人，以及管理頻道的秩序，有些機器人甚至可以[播音樂](https://docs.yeecord.com/guide/commands/music/#play)或是跟他玩小遊戲！

你不需要成為一個程式天才也沒關係！你也可以使用現成的機器人範本，並且修改它，或是到 [top.gg](https://top.gg) 和 [Discord Bot List](https://discord.bots.gg) 上尋找喜歡的機器人，[YEE 式機器龍](https://yeecord.com/) 也是一個不錯的選擇

<Adsense />

### 音樂機器人

音樂機器人是在 Discord 上比較特別的一種形式，透過在文字頻道輸入指令，
讓機器人到你的語音頻道播放音樂，而且他讓許多的成員能夠同時一起收聽一樣的音樂，
這也是一種很好的方式讓你的伺服器更加的穩定，而且也讓你的頻道更加的熱門

### Rythm 和 Groovy 被下架

在 2021 年 8 月的時候，Discord 上大型音樂機器人 Groovy 作者 Nik 在他的群組上表示收到來自 Google 的信件，在信中表示，Groovy 因違反了「YouTube 服務條款」，而要他們停止提供服務，Nik 也在公告上表示他們將在 2021 年 8 月 30 號 終止服務

![Groovy擁有者表示他的機器人將被下架](@static/blog/how-to-invite-bot/groovy-down.png)

而同樣的，正當大家正在猜測 Rythm 會不會是下一個受害者的時候，Rythm 也確定了在 9 月 15 號 終止了他們的服務

## 邀請 YEE 式機器龍

1. 開啟連結 https://yeecord.com/link

   ![邀請機器人](@static/blog/how-to-invite-bot/invite-bot-1.png)

2. 選擇目標伺服器

   <Tip title="有用的小提示">
     如果有遇到下方出現紅字錯誤說要驗證 Email，請參照
     [邀請故障排解](/blog/discord-bot-troubleshooting/#%E9%9C%80%E8%A6%81%E9%A9%97%E8%AD%89-email)
     的說明。
   </Tip>

3. 點授權 提供你的權限來邀請機器人 (盡量不要動到機器人權限避免無法使用)

   ![點擊授權](@static/blog/how-to-invite-bot/invite-bot-2.png)

這樣就成功邀請我到你的伺服器了，回到 Discord 應該會發現我出現在成員清單裡，如果要查看我的使用方法請到 [YEE 式機器龍文檔](/docs/)，或是點擊左上角 `使用教學`。

4. 點選你的伺服器左邊的任一個語音頻道進去

   ![Discord 語音頻道](@static/blog/how-to-invite-bot/voice-channel.png)

   如果左邊沒有語音頻道可以點選伺服器名稱，然後建立頻道

   ![Discord 建立頻道](@static/blog/how-to-invite-bot/create-channel.png)

5. 在文字頻道輸入指令來播放音樂

   ![播放音樂](@static/blog/how-to-invite-bot/play-music.jpg)

   <Tip title="有用的小提示">
     如果遇到機器人沒反應、沒聲音、錯誤訊息等等，請繼續閱讀這篇：[《Discord
     音樂機器人沒聲音、錯誤、無法邀請？「5
     方法」排解機器人問題！》](/blog/discord-bot-troubleshooting/)
   </Tip>

## 同場加映：如何調整音量

在 YEE 式機器龍 移除音量調控功能後，相信許多人都對音量控制霧煞煞，不知道如何調整
但其實 Discord 就內建有調整音量的功能，只要對著機器人點擊右鍵並拖動旋扭就可以了！

![Discord 調整 音量](@static/blog/how-to-invite-bot/adjust_volume.png)

## YEE 式機器龍指令

以下是我整理出來的**YEE 式機器龍音樂指令**以及介紹。

| 指令描述                   | YEE 式機器龍指令                               |
| -------------------------- | ---------------------------------------------- |
| 輸入關鍵字搜尋一首音樂     | &p 我把我的青春給你                            |
| 播放 YouTube URL           | &p https://www.youtube.com/watch?v=iJAxeafvn8Y |
| 開啟語音頻道小活動         | &yt                                            |
| 啟用 / 關閉 Nightcore 效果 | &effect nightcore / &effect off                |
| 查看等候播放的清單         | &q                                             |
| 顯示正在播放的音樂         | &np                                            |
| 跳過目前的音樂             | &s                                             |
| 暫停播放                   | &pause                                         |
| 繼續播放                   | &resume                                        |
| 中斷機器人的音樂           | &leave                                         |
| 讓機器人加入語音頻道       | &join                                          |
| 隨機打亂播放清單           | &shuffle                                       |
