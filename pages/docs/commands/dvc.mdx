# 動態語音頻道

import Adsense from "@components/mdx/Adsense";

## 什麼是動態語音

<Adsense />

Discord 有提供文字頻道以及語音頻道讓我們打造自己的伺服器，但是為了要為不同的用途創建頻道，像是 `遊戲用`、`聽音樂用` 等等，當頻道一多的時候，看起來就會很混亂。

![語音頻道 分類](@static/docs/dvc/voice-channel.png)

而透過使用**動態語音系統**，你只需要一個頻道就可以滿足各個人的需求！當有成員加入那個語音頻道的時候，會創建一個專屬於他的頻道，而其他人也可以加入他的頻道，當頻道內沒有人的時候會自動收回！

## 初始化動態語音

先注意為了避免濫用此功能，使用指令的人必須要有 `管理頻道` 的權限，並且機器人要有 `管理頻道`、`連結到語音頻道`、`移動成員` 的權限，才能正常運作！

<Adsense />

要開始使用動態語音輸入 `&dvc` 之後選擇 `初始化` 就會生出一個頻道叫做 `創建頻道`，第一次進去可能會需要一點時間讓機器人更新資料，之後就可以開始使用了！

![動態語音頻道](@static/docs/dvc/dvc-command.png)

## 頻道放在同一個類別下面會被刪掉嗎?

非生出來的頻道不會被刪掉！動態語音是辨識頻道的名字格式，所以請不要去改生出來的頻道的名字，不然他會判定是其他的頻道哦
