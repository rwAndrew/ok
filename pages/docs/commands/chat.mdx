## 什麼是跨群聊天

<Adsense />

因為 Discord 限制各位每個最多能夠加入 `100` 個伺服器，擁有 Discord Nitro 則可以加入 `200` 個伺服器，但在各個群組不斷增長之下，這個限制似乎已經不夠用了，跨群聊天讓你不用再額外的加入別的伺服器，就能和別的伺服器的人交流。

## 那我要怎麼開始?

先注意為了避免濫用此功能，使用指令的人必須要有 `管理頻道`、`管理Webhooks`、`管理訊息` 的權限，並且機器人要有 `管理Webhooks`、`管理訊息` 的權限，才能正常運作！

使用 `&chat create 名字` 指令建立一個新的聊天室，聊天室名稱可以自訂，但**不能重複**，並且**只能使用英數**，創建好之後就可以往下看 [如何邀請其他人](#加入聊天室)。

![創建 跨群聊天室](@static/docs/chat/chat-create.png)

<Adsense />

## 加入聊天室

使用 `&chat join 名字` 指令加入一個已經存在的聊天室，為避免有人濫用我們的功能，一個伺服器只能加入一個跨群聊天群組，當加入了聊天之後，我們會在你的頻道創建一個 Webhook 供我們發送訊息使用，請不要把他刪除！

同樣的，退出聊天室使用 `&chat leave` 指令

## 我怎麼收不到訊息?

請使用 `&chat` 指令確認目前有正確的連接到任何跨群聊天室，如果沒有連結過請看 [如何加入聊天室](#加入聊天室)，如果加入後還是沒反應，請確認機器人其他指令是否正常運作 (如 `&help`)，可能正在重新啟動或是進行維護。

如果仍然有問題，請到 [支援伺服器](https://yeecord.com/support) 告訴我們！

![跨群聊天室 資訊](@static/docs/chat/chat-info.png)

## 如何使某人變成跨群管理員？

使用 `&chat op @使用者`

## 如果有人不遵守你所訂的跨群規則？

首先使用 `&chat minfo 訊息ID` 來查詢使用者 ID

![跨群聊天室 訊息ID](https://media.discordapp.net/attachments/856382888089747520/988043525578977290/IMG_2393.png)

複製在成員下方的一長串數字，之後永久禁言那個用戶打 `&chat ban 用戶ID 原因` 用戶的 ID 就是我們剛剛複製的 ID

![跨群聊天室 禁言](https://media.discordapp.net/attachments/856382888089747520/988046033684676688/IMG_2394.png)

<Warning>
執行此指令的時候必須是跨群管理員或是擁有者，不然會顯示 `❌|你沒有權限這麼做`，還有一旦永久禁言了就無法解除請再三確認你是否打對 ID

另外在伺服器內身份組也必須要有 `管理頻道` 還有 `管理訊息` 的權限，否則真的會很尷尬喔！

</Warning>
