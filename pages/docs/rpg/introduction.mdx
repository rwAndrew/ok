# RPG 遊戲簡介

import { Adsense, Info } from "@components/mdx";

歡迎來到我們的角色扮演遊戲世界！讓我們開始吧~

## 去異界存八萬金

<div className="steps-container">

### 取得工作

因為現在你是沒有錢的狀態，所以首先來到這個世界第一件事情是取得一份工作！先使用 `&job` 指令在下拉選單裡面選擇一個你喜歡的職業，詳細的工作說明可以到 [民生工作](./jobs) 查看。

如果要轉職的話，可以再次使用 `&job` 指令並且選擇新的職業，選擇職業以及轉職都會有 168 小時的 CD，請謹慎轉職！若您選擇轉職成與原先相同的職業也會視為轉職並進入 CD

### 開始工作

當你有了工作之後就可以開始工作！~~心臓を捧げよ！！~~ 參考 [民生工作](/docs/rpg/jobs) 來工作並獲得收益，每次工作會消耗 `1` 點體力值，當你沒體力的時候就可以 [吃東西](#吃東西)。

工作會有 CD，工作次數越多次 CD 時間會越長，每天 0 時重置工作次數以及 CD 時間

### 每日簽到

每天到我們的官方伺服器聊天的時候，聰明的你應該有發現訊息下方出現一個 `💰` 錢錢的符號，那就表示你有成功簽到了哦！

簽到會回復 `3 ~ 7` 體力，並且給予每天的獎金，請記得不要錯過

### 幫我們投票

每 12 個小時可以到 [top.gg](https://yeecord.com/topgg) 投票，並且每次獲得 `300$` 的獎勵，另外也可以到 [DiscordTw](https://yeecord.com/dst) 投票，並且每次獲得 `500$` 的獎勵。

</div>

## 吃東西

工作了一整天，終於可以吃飯了 ~~敲桌~~，但是！天下可沒有白吃的午餐，找尋其他玩家商店內有沒有食物(請等待其他玩家使用 `&shop list` 展示他的商店後再行購買) [商店](#商店) 購買你要吃的食物

購買到之後可以用 `&eat` 吃下肚，如果是生的食物可以交給 [廚師](/docs/rpg/jobs#燒烤) 使用 `&bake` 指令把它烤熟！當然你需要自己到商店準備煤炭，每個煤炭可以烤 `2` 個食物。

## 商店

這個世界的商店非常神奇，什麼都有賣！等待玩家使用 `&shop list` 展示他們的商店後，使用 `&buy` 指令向其他玩家購買物品，確認後即可完成購買，您會獲得對應的物品並扣除相對應的錢給賣方 [商店](/docs/rpg/shop/)。

<Adsense />

而要購買則是使用 `&buy` 指令，很抱歉我們沒有提供 7 天鑑賞期，買了就不能退貨了哦！

## 應對孤獨

一個人玩太無聊？我們有人際關係系統！

<div className='steps-container'>

### 夥伴

在這個世界有著共同的夥伴！先講講成為夥伴可以使用 `&partner` 指令，請注意，被邀請擔任夥伴者一次只能有一個夥伴，簡單來說就是一個人只能被一間公司聘用，而老闆可以聘用很多員工

當然夥伴體力不足的時候你也可以負責餵食牠，使用 `&feed` 指令餵食你的食物給他

### 結婚

當你終於在這世界找到了合適的伴侶，而你覺得正是時機向她求婚的時候，使用 `&marry` 指令來向你喜歡的對象求婚，也祝你們有美好的新生活（灑花

</div>

## 規則

### 使用條款

1.RPG 遊戲請使用人工進行指令操作，切勿使用機器人腳本等(例如: auto360 加速器等)，若被檢舉或偵測到使用機器人腳本證實者將被永久 ban 帳

2.請勿使用任何漏洞或修改參數等不法行為來獲取不正當利益，被檢舉或被查驗證實者將永久 ban 帳

3.試圖詐騙或詐騙者經檢舉並查核後將被永久 ban 帳

4.管理層保有對 RPG 最終解釋以及管理權，以最高權限原則

5.各伺服器對於 RPG 規則請依照伺服器擁有者規範執行，前提是所有 RPG 使用者必須遵守使用條款，違反者將會面臨懲處

### Yeecord RPG 規則

1.在除 RPG 指令以外的頻道使用 RPG 或其他一般機器龍指令者予以禁言 1 天

2.請勿使用 `&shop`、`&buy`、`&store` 等查看他人商店或在跟對方沒有共識情況下使用 `&marry` 指令以及使用 `&money` 查看他人的交易紀錄，經查核並被當事人檢舉將會予以 1 天禁言（非當事人請勿帶替他人檢舉），若要查看他人商店請使用 ID 查詢。ID 查詢方式：在設定開起開發者模式，即可在使用者個人資料最下方（手機）或者點擊滑鼠右鍵（電腦）複製 ID

3.騷擾行為、使用 `&av` 或私訊騷擾伺服器成員，經當事人檢舉核實後將予以嚴重處分

4.若檢舉後發現檢舉內容不實導致被檢舉人承受不應有的懲罰，予以檢舉人 7 天禁言

5.管理層保有對 RPG 最終解釋以及管理權，以最高權限原則
